Proyecto Argentina Programa

Desarrollo de una aplicación web full stack, que muestra: los datos personales, estudios cursados, experiencia laboral y conocimiento de las tecnología. Es decir, ¡crear mi propio portfolio web! Esta aplicación es de arquitectura distribuida y contiene un diseño de interfaz de usuario (front end) que muestra la información, una base de datos que almacene los datos antes mencionados y cuenta con las APIs necesarias para proveer a través de internet la información (back end). La idea, además de servir para aplicar lo aprendido , es que pueda publicar el portfolio web y usarlo como carta de presentación en el mercado laboral.
